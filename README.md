# 视频压缩工具
基于 arrdio + Vue + FFmpeg 开发的本地桌面视频处理工具，仅在本地运行，所有文件处理均不经过云端，全程本地加密处理，安全无上传。

## ✨ 功能
- 一键视频压缩，大幅度减小视频体积，兼顾画质与大小
- 自定义压缩参数、分辨率、码率调节
- 批量视频处理
- 格式转换、视频裁剪基础功能
- 无广告、无后台、无网络请求、纯本地运行

## 📦 环境与运行
### 环境要求
- Windows 10 及以上系统
- 无需安装 FFmpeg，程序内置封装

### 运行方式
1. 下载 Release 发布包
2. 直接解压运行 exe 程序即可使用
